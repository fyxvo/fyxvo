# Fyxvo Launch Assets

## 1. Launch Announcement

Fyxvo is live on Solana devnet. Teams can connect a wallet, activate a project on chain, fund it with SOL, generate an API key, and send real JSON-RPC traffic through the hosted relay while watching analytics and status surfaces react in real time.

## 2. Product Summary

Fyxvo is a funded Solana RPC control plane built around a clean first-user path. It combines wallet-authenticated project control, on-chain project activation, SOL treasury funding, standard and priority relay routing, request logging, worker rollups, and honest public status surfaces in one product.

## 3. Feature Summary

1. Wallet-authenticated project control with no private key storage.
2. On-chain project activation and SOL funding on Solana devnet.
3. Hosted standard RPC and priority relay endpoints with clear pricing.
4. Gateway balance enforcement, request logging, and worker-driven analytics.
5. Public status surfaces that state what is live, what is gated, and what is managed.

## 7. X Thread Draft

1. Fyxvo is live on Solana devnet. The goal is simple: make the first real RPC session feel structured instead of fragmented.

2. You connect a wallet, sign a challenge, activate a project on chain, fund it with SOL, generate an API key, and send traffic through hosted relay endpoints.

3. The standard RPC path and priority relay path are separate on purpose. Pricing, routing, and rate behavior stay explicit instead of being hidden behind one vague endpoint.

4. The public status surface is meant to be useful, not ornamental. It shows hosted service health, protocol readiness, what is live today, and what is still intentionally gated.

5. SOL is the live public funding path on devnet today. USDC is still configuration-gated until it is explicitly enabled and validated for the deployment.

6. The current launch topology is managed infrastructure first. Fyxvo says that plainly instead of pretending the operator supply is already wide open.

7. If you want to try the path yourself, start here: https://www.fyxvo.com/docs
